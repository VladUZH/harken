# Bundled fonts

These fonts are bundled locally so the dashboard works fully offline with no CDN —
in keeping with Harken's "no telemetry, data never leaves your box" premise.

All three are licensed under the **SIL Open Font License 1.1** (OFL), which permits
bundling and redistribution:

- **Fraunces** — Undercase Type / Phaedra Charles. https://github.com/undercasetype/Fraunces
- **Hanken Grotesk** — Alfredo Marco Pradil. https://github.com/marcologous/hanken-grotesk
- **JetBrains Mono** — JetBrains. https://github.com/JetBrains/JetBrainsMono

woff2 files retrieved via [Fontsource](https://fontsource.org/) (latin subsets).
See each project for the full OFL text.
